Building Blocks

Manifest File

JSON file that provides an extension's metainformation (goo.gl/2otQP).

manifest.json

{
  "name": "A Sample Extension",
  "version": "0.0.1",
  "description": "This is a sample extension.",
  "icons": {
    "128": "icon_128.png"
  },
  "browser_action": {
    "default_icon": "icon_19.png",
    "default_popup": "popup.html"
  },
  "background_page": "background.html",
  "permissions": [ "notifications", "geolocation" ]
  …
}

Background Page

HTML document that runs JavaScript persistently and invisibly (goo.gl/EwiKy).

Popup Page

HTML document with which the user interacts (goo.gl/M6Oe3).

Modern JavaScript APIs

WebKit ♥ Desktop notifications, geolocation, local storage mechanisms, getters/setters, etc.

chrome.* APIs

The current list of stable extension APIs is at goo.gl/lTzWn

Make sure you're running the latest dev version of Chrome.

Experimental APIs are also available (goo.gl/Kn7vd), but are locked behind a flag.
